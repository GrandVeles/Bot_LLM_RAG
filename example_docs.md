# Руководство пользователя — Пример документа техподдержки

## 1. Установка программного обеспечения

Для установки системы выполните следующие шаги:

1. Скачайте дистрибутив с официального сайта.
2. Запустите установщик от имени администратора.
3. Следуйте инструкциям мастера установки.
4. После установки перезагрузите компьютер.

### Системные требования

- Операционная система: Windows 10/11 или Linux Ubuntu 20.04+
- Оперативная память: не менее 4 GB
- Свободное место на диске: 10 GB

## 2. Часто задаваемые вопросы (FAQ)

### Как сбросить пароль?

Для сброса пароля:
1. Перейдите на страницу входа.
2. Нажмите «Забыли пароль?».
3. Введите ваш email.
4. Следуйте инструкции в письме.

### Программа не запускается

Если программа не запускается, проверьте:
- Наличие актуальной версии .NET Framework (Windows).
- Права доступа к папке установки.
- Наличие блокировки антивирусом.

### Как обновить программу?

Обновление происходит автоматически при запуске. Для ручного обновления:
- Откройте меню «Справка» → «Проверить обновления».

## 3. Устранение типичных неполадок

### Ошибка подключения к базе данных

**Симптом:** сообщение «Connection timeout» при запуске.

**Решение:**
1. Убедитесь, что служба базы данных запущена.
2. Проверьте настройки файрвола — порт 5432 (PostgreSQL) должен быть открыт.
3. Проверьте строку подключения в файле `config.ini`.

### Медленная работа системы

**Причины и решения:**
- Недостаточно RAM → закройте лишние приложения или увеличьте объём памяти.
- Фрагментация диска → выполните дефрагментацию (только HDD).
- Большой объём логов → очистите папку `/var/log/app/` или `C:\App\Logs\`.
